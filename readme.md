# Three.js Journey

## Overview
This project is a learning exercise for Three.js and JavaScript. It demonstrates the use of Three.js for creating interactive 3D scenes, along with modern frontend practices using Vite and Vercel for deployment.

## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run the following commands:

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```

## Project Structure
- **src/**: Contains the main source code.
  - **Experience/**: Core application logic, including 3D scene management, utilities, and world objects.
  - **index.html, script.js, style.css**: Entry points for the application.
- **static/**: Static assets such as sounds, textures, environment maps, and favicons.
- **Configuration**: Uses Vite for development and Vercel for deployment.
